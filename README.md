# Sleight & Co.

Concept homepage for Sleight & Co. Group, a premium umbrella brand for marketing,
technology, hospitality, beauty, and owned venture work.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
